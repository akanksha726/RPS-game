## 🎲 Game Rules

1. You choose **Rock**, **Paper**, or **Scissors**.
2. The computer randomly picks one as well.
3. The result is displayed:
   - 🪨 Rock crushes Scissors  
   - 📄 Paper covers Rock  
   - ✂️ Scissors cuts Paper
